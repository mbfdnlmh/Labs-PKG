Документация пользователя для лабораторной 1 "Конвертер цветовых моделей"
1. Введение

Приложение Конвертер цветовых моделей позволяет пользователю преобразовывать цветовые значения между тремя основными цветовыми моделями: RGB, XYZ и CMYK. Программа предоставляет простой и удобный интерфейс для работы с цветами, поддерживает как ввод значений вручную, так и выбор цвета с помощью диалогового окна.
2. Запуск приложения

    Скачайте и установите приложение.
    Запустите программу через ярлык или исполнимый файл.

3. Основные элементы интерфейса

    Текстовые поля (QLineEdit) — для ввода значений каждого компонента цвета в моделях RGB, XYZ и CMYK.
    Ползунки (QSlider) — для изменения значений компонентов цвета. Значение ползунка автоматически обновляет соответствующее текстовое поле.
    Кнопка "Выбрать цвет" — открывает диалоговое окно для выбора цвета. После выбора цвета значения RGB автоматически обновляются.

4. Использование программы

    Изменение значений через текстовые поля: Введите значение для компонента цвета в соответствующем текстовом поле (например, для R, G, B в RGB). После ввода нажмите Enter. Программа автоматически выполнит конвертацию и обновит другие значения в интерфейсе.

    Изменение значений через ползунки: Перетащите ползунок для изменения значений компонента цвета. Значения ползунков автоматически отобразятся в текстовых полях.

    Выбор цвета: Нажмите на кнопку "Выбрать цвет" для открытия диалогового окна выбора цвета. Выберите цвет из палитры, и значения его компонентов (RGB) автоматически отобразятся в текстовых полях.

    Преобразование между моделями: Программа автоматически конвертирует введенные значения цвета в другие модели (например, RGB → XYZ или RGB → CMYK) и обновляет все соответствующие текстовые поля и ползунки.

5. Предупреждения

Если вы введете некорректное значение (например, RGB значения, превышающие диапазон от 0 до 255), программа покажет предупреждение. Убедитесь, что значения находятся в допустимых пределах:

    RGB: от 0 до 255.
    XYZ: от 0 до 100.
    CMYK: от 0 до 100.

6. Заключение

Конвертер цветовых моделей — это удобный инструмент для работы с цветами и их преобразования между популярными моделями. Программа может быть полезна в различных областях, таких как графический дизайн, печать и цифровая обработка изображений.

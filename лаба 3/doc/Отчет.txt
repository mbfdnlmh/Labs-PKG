Отчет
1. Описание проекта

Данное приложение представляет собой инструмент для обработки изображений, который позволяет пользователю применять различные алгоритмы фильтрации и порогового преобразования изображений. Программа использует библиотеку OpenCV для обработки изображений и Tkinter для создания графического интерфейса. Пользователи могут загружать изображения, применять к ним фильтры и пороги, а затем сохранять обработанные результаты.
2. Описание функций

Основные функции программы:

    Функции обработки изображений:

        apply_highpass_filter(image)
        Применяет фильтр высоких частот, который улучшает резкость изображения, используя ядро размером 3x3.

        apply_local_threshold(image, method="mean")
        Применяет локальный порог с двумя методами: "mean" (средний порог) и "gaussian" (гауссов порог). Метод "mean" использует среднее значение пикселей для определения порога, а метод "gaussian" использует гауссовое распределение для вычисления порога.

        apply_global_threshold(image, threshold_value=128)
        Применяет глобальный порог с фиксированным значением, превращая изображение в черно-белое, где значения выше порога становятся белыми, а ниже — черными.

        apply_adaptive_threshold(image)
        Применяет адаптивный порог, который вычисляет порог для каждой области изображения на основе местных характеристик.

    Функции взаимодействия с графическим интерфейсом:

        open_image()
        Открывает диалоговое окно для выбора изображения с помощью файлового менеджера и отображает его на экране.

        display_image(image)
        Отображает выбранное или обработанное изображение в графическом интерфейсе с использованием Tkinter.

        process_highpass(), process_local_threshold_mean(), process_local_threshold_gaussian(), process_global_threshold(), process_adaptive_threshold()
        Каждая из этих функций применяет соответствующую обработку изображения (например, фильтрация или пороговые методы) и обновляет отображаемое изображение на экране.

        save_image()
        Сохраняет обработанное изображение в формате PNG по выбору пользователя.

3. Архитектура и дизайн приложения

Приложение состоит из нескольких модулей:

    Tkinter: отвечает за создание и управление графическим интерфейсом. Включает в себя элементы управления, такие как кнопки для загрузки изображений, применения фильтров и порогов, а также сохранения изображений.

    OpenCV: используется для обработки изображений. Все операции по изменению изображений (фильтрация, пороговое преобразование) выполняются с помощью этой библиотеки.

    Pillow (PIL): используется для преобразования изображений из формата OpenCV в формат, совместимый с Tkinter для отображения.

4. Потоки данных

    Загрузка изображения: Пользователь нажимает кнопку "Открыть изображение", после чего выбирает файл на своем компьютере. Изображение загружается в память программы.

    Обработка изображения: После загрузки изображения, пользователь может применить различные фильтры и пороговые преобразования. Результат обработки отображается в окне приложения.

    Сохранение изображения: Пользователь может сохранить обработанное изображение в выбранном формате (PNG).

5. Проблемы и решения

    Проблема: При отображении изображения через Tkinter требуется преобразование изображения из формата BGR в формат RGB (OpenCV использует BGR по умолчанию).
    Решение: Использована функция cv2.cvtColor() для преобразования изображения в формат RGB перед отображением его с помощью Tkinter.

    Проблема: Нужно быстро обрабатывать изображения без потери производительности.
    Решение: Применение оптимизированных операций с использованием библиотеки NumPy для векторизации некоторых процедур.

6. Будущие улучшения

    Поддержка других фильтров и преобразований: Расширение функционала программы путем добавления дополнительных фильтров и методов обработки изображений.
    Гибкость настроек: Возможность изменять параметры фильтров, например, размер ядра для фильтрации или размер окна для локальных порогов.
    Поддержка дополнительных форматов: Добавление поддержки других форматов изображений для загрузки и сохранения.

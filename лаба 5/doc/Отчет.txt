Отчет по лабораторной 5 "Алгоритм отсечения отрезков и многоугольников"

1. Введение

Целью данного проекта является разработка графического интерфейса для демонстрации алгоритмов отсечения отрезков и многоугольников. Программа реализует два алгоритма:

    Алгоритм Сазерленда-Коэна для отсечения отрезков прямых, выходящих за пределы заданного прямоугольного окна.
    Алгоритм Сазерленда-Ходжмана для отсечения многоугольников, пересекающихся с прямоугольным окном.

2. Описание программы

Программа состоит из следующего:

    Возможности для загрузки данных из текстового файла, содержащего сегменты или многоугольники.
    Алгоритмы для отсечения отрезков и многоугольников в зависимости от типа данных.
    Графический интерфейс, отображающий результат работы алгоритмов.

3. Алгоритмы

    Сазерленда-Коэна: Используется для отсечения отрезков прямых, выходящих за пределы прямоугольного окна. Каждый отрезок проверяется на принадлежность окну и, если необходимо, отсекается.
    Сазерленда-Ходжмана: Используется для отсечения выпуклых многоугольников, которые пересекаются с прямоугольным окном. Многоугольник отсеивается поочередно по каждой из четырех граней окна.

4. Результаты

Программа успешно отсекала как отрезки, так и многоугольники, которые выходили за пределы заданного окна. Для каждого типа данных пользователю предложены соответствующие алгоритмы отсечения. Результаты работы алгоритмов визуализируются на графике, где показаны как исходные объекты, так и их отсеченные версии.

5. Заключение

Проект успешно продемонстрировал использование алгоритмов отсечения в графических приложениях, а также предоставил удобный инструмент для их визуализации с использованием Python и Tkinter.

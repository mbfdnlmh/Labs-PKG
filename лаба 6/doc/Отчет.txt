Отчет по лабораторной 6: 3D-визуализация буквы X

Введение: Данный проект представляет собой создание интерфейса для 3D-визуализации буквы "X" с возможностью применения различных преобразований, таких как масштабирование, перенос, вращение и проекции на координатные плоскости. Программа реализована с использованием библиотеки Tkinter для графического интерфейса и Matplotlib для построения графиков.

Цель проекта: Целью является разработка инструмента, который позволит пользователям взаимодействовать с 3D-объектом буквы "X" с помощью интерфейса, реализующего базовые операции над 3D-геометрическими объектами.

Описание функциональности:

    Программа предоставляет несколько кнопок для выполнения различных операций:
        Масштабирование
        Перенос
        Вращение по осям X, Y, Z
        Проекции на плоскости XY, XZ, YZ
        Сброс до исходного состояния

Каждое из преобразований осуществляется с помощью соответствующих матриц преобразования. Все действия сопровождаются визуализацией результата на 3D-графике.

Реализация:

    Масштабирование: Осуществляется с помощью матрицы масштабирования, которая изменяет размеры буквы.
    Перенос: Процесс сдвига объектов по осям X, Y, Z.
    Вращение: Осуществляется вращение объекта вокруг одной из координатных осей с заданным углом.
    Проекции: Создаются 2D-проекции объекта на различные координатные плоскости.
    Сброс: Версия, возвращающая объект в исходное положение.

Заключение: Проект успешно реализует функциональность, позволяющую управлять 3D-объектом буквы "X". Пользователи могут изменять параметры объекта в реальном времени, что полезно для учебных целей, например, при изучении геометрических преобразований в трехмерном пространстве.
